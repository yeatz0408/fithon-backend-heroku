# Toshokan - FitHon
## _運動する人のための図書館_ (ジョンノア、1人開発)

FitHonは主にMySql、Java、Spring Boot、JavaScript、React.jsをベースに開発されました。また、Bootstrapを使用してモバイルでも快適に利用できるように設定されています。

## 機能

- 完全なＣＲＵＤ機能
- OKTAAuthenticationでアドミンと一般ユーザーで別れるUI
- 各ユーザーごとに最大5冊の貸し出し、返却、延長し、図書を管理する
- 各ユーザーごとして読んだ本のレビューを残して過去に貸与した本を閲覧
- アドミンとして新しい本を登録したり、貸与可能な本の数を変更
- MySql、Spring Rest と React.js の個別のエンドポイント
- Bootstrap 5を利用したシンプルなデザイン
- モバイル対応可能
